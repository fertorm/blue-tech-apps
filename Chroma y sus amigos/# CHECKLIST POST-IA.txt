#### Semana 5-6: Sistema de Audio y Feedback
**Objetivo**: Hacer la app más inmersiva para niños

**Features a implementar:**
- AudioService.ts (servicio centralizado)
- Sonidos de feedback positivo
- Música de fondo opcional
- Voces en off (nombres de colores)

**Prompt para IA:**
```
[Pegar PROJECT_BRAIN.md]

Necesito implementar sistema de audio para app infantil.

REQUISITOS:
- AudioService.ts que maneje todos los sonidos
- Usar Web Audio API (no <audio> tags)
- Precarga de sonidos para evitar delay
- Control de volumen global
- Método play() que nunca falle (try/catch)

SONIDOS NECESARIOS:
- success.mp3 (cuando acierta)
- tap.mp3 (cuando toca botón)
- complete.mp3 (cuando completa actividad)

¿Dónde consigo sonidos free? ¿Cómo los integro?
```

**Recursos que la IA te sugerirá:**
- Freesound.org
- Zapsplat.com
- ElevenLabs (voces AI gratis)

### 🟡 MES 3-4: EXPANSIÓN DE CONTENIDO (Semanas 9-16)

#### Semana 9-10: Juego 3 - "Encuentra el Color"
**Concepto pedagógico**: Reconocimiento de colores en contexto

**Prompt para IA:**
```
[Pegar PROJECT_BRAIN.md]

NUEVO JUEGO: "Encuentra el Color"

MECÁNICA:
- Voz dice "Encuentra el rojo"
- Pantalla muestra 6 objetos de diferentes colores
- Niño toca el objeto del color correcto
- Feedback positivo + siguiente color

PROGRESIÓN:
- Nivel 1: 3 objetos, colores primarios
- Nivel 2: 4 objetos, incluye secundarios
- Nivel 3: 6 objetos, incluye tonos (rojo claro/oscuro)

IMPLEMENTACIÓN:
- Componente FindColorGame.tsx
- Integrar AudioService para voz
- Usar animaciones existentes
- Guardar progreso localmente (localStorage)

Dame código completo + assets necesarios
```

---

#### Semana 11-12: Juego 4 - "Mezcla de Colores"
**Concepto pedagógico**: Teoría del color básica

**Prompt para IA:**
```
[Pegar PROJECT_BRAIN.md]

NUEVO JUEGO: "Mezcla Mágica"

MECÁNICA:
- 2 botes de pintura (ej: amarillo + azul)
- Niño arrastra uno sobre otro
- Animación de mezcla
- Aparece nuevo color (verde)
- Celebración

FÍSICA:
- Colores primarios: rojo, amarillo, azul
- Mezclas: rojo+amarillo=naranja, azul+amarillo=verde, rojo+azul=morado

TÉCNICO:
- Drag & drop touch-friendly
- Animación de líquido mezclándose
- ColorMixingService que calcule resultado
- Tutorial interactivo la primera vez

¿Cómo implemento drag&drop accesible para niños de 3 años?
```

#### Semana 13-14: Juego 5 - "Memoria de Colores"
**Prompt para IA:**

MECÁNICA:
